§ 1. Installing Pliik

§ 1.1. CouchDB

Pliik is a Couchapp served directly from a CouchDB instance.
Both the source code and the application data are stored at the persistence layer.

First step is to get a CouchDB instance up an running.
You can choose to:

  • Install your own CouchDB instance - CouchDB Instalation (>=1.1.x)

  • … or to get one from the web at Iriscouch
