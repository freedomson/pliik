\section{Testing Counters}
\newcounter{foo}
Foo is \thefoo.

Increment foo.
\stepcounter{foo}
Now Foo is  \thefoo.


Set Foo to 2.
\setcounter{foo}{2}
Now Foo is  \thefoo.

Add ten to Foo.
\addtocounter{foo}{10}
Now Foo is  \thefoo.

Double Foo
\addtocounter{foo}{\value{foo}}
Now Foo is  \thefoo.

\section{Testing RefStep}
Define bar to be reset within foo.
\newcounter{bar}[foo]
Now (Foo,Bar) is (\thefoo,\thebar)

Refstep bar: \refstepcounter{bar}
Now (Foo,Bar) is (\thefoo,\thebar)

Refstep foo: \refstepcounter{foo}
Now (Foo,Bar) is (\thefoo,\thebar)
